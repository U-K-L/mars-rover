168  /*
169  * Register a periodic callback for this bus. All callbacks on the
170  * same bus are made from the same thread with lock already taken. In
171  * other words, the callback is not executed on the main thread (or the
172  * thread which registered the callback), but in a separate per-bus
173  * thread.
174  *
175  * After registering the periodic callback, the other functions should not
176  * be used anymore from other contexts. If it really needs to be done, the
177  * lock must be taken.
178  *
179  * Return: A handle for this periodic callback. To cancel the callback
180  * call #unregister_callback() or return false on the callback.
181  */
