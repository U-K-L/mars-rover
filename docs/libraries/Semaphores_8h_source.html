24 /*
25  a method to make semaphores less error prone. The WITH_SEMAPHORE()
26  macro will block forever for a semaphore, and will automatically
27  release the semaphore when it goes out of scope
28 
29  Note that we have two types of semaphores. A normal semaphore can
30  only be taken once. A recursive semaphore allows for the thread
31  holding the semaphore to take it again. It must be released the same
32  number of times it is taken.
33 
34  The WITH_SEMAPHORE() macro can be used with either type of semaphore
35  */
