59 /*
60  * Register a periodic read. This should be called after the slave sensor is
61  * already configured and the only thing the master needs to do is to copy a
62  * set of registers from the slave to its own registers.
63  *
64  * The sample rate is hard-coded, depending on the sensor that exports this
65  * AuxiliaryBus.
66  *
67  * After this call the AuxiliaryBusSlave is owned by this object and should
68  * not be destroyed. A typical call chain to use a sensor in an AuxiliaryBus
69  * is (error checking omitted for brevity):
70  *
71  * AuxiliaryBusSlave *slave = bus->request_next_slave(addr);
72  * slave->passthrough_read(WHO_AM_I, buf, 1);
73  * slave->passthrough_write(...);
74  * slave->passthrough_write(...);
75  * ...
76  * bus->register_periodic_read(slave, SAMPLE_START_REG, SAMPLE_SIZE);
77  *
78  * @slave: the AuxiliaryBusSlave already configured to be in continuous mode
79  * @reg: the first register of the block to use in each periodic transfer
80  * @size: the block size, usually the size of the sample multiplied by the
