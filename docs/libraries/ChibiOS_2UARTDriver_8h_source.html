93  /*
94  return timestamp estimate in microseconds for when the start of
95  a nbytes packet arrived on the uart. This should be treated as a
96  time constraint, not an exact time. It is guaranteed that the
97  packet did not start being received after this time, but it
98  could have been in a system buffer before the returned time.
99 
100  This takes account of the baudrate of the link. For transports
101  that have no baudrate (such as USB) the time estimate may be
102  less accurate.
103 
104  A return value of zero means the HAL does not support this API
105  */
