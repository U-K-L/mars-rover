67  /*
68  * A compass measurement is expected to pass through the following functions:
69  * 1. rotate_field - this rotates the measurement in-place from sensor frame
70  * to body frame
71  * 2. publish_raw_field - this provides an uncorrected point-sample for
72  * calibration libraries
73  * 3. correct_field - this corrects the measurement in-place for hard iron,
74  * soft iron, motor interference, and non-orthagonality errors
75  * 4. publish_filtered_field - legacy filtered magnetic field
76  *
77  * All those functions expect the mag field to be in milligauss.
78  */
